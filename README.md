*Линтеры:* Stylelint, Eslint, Prettier
## Запуск проекта

Клонирование проекта

```bash
  git clone https://github.com/Barry-Kenan/top-app.git
```

Переходим в папку с проектом

```bash
  cd top-app
```

Установка зависимостей

```bash
  npm install
```

Запуск проекта локально 

```bash
  npm run dev
```
